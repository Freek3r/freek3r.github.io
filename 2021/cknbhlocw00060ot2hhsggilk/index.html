PHPStudy安装与使用

安装

  1. 下载 PHPStudy

  2. 点击自定义选项可以手动选择安装路径，但一定要注意：路径不能有中文

  3. 安装完成

使用

  1. phpstudy 各功能模块简介
  2. 利用phpstudy在本地搭建网站

DVWA（DAMN VULNERABLE WEB APPLICATION ）

  1. 从DVWA官网或GitHub下载或克隆文件到本地

  2. 在 PHPStudy 中新建网站，并将 DVWA 解压到网站根目录

    最终结果应该如图所示

  3. 将 config 目录下的 config.inc.php.dist 复制并改名为 congfig.inc.php ，然后用记事本打开并编辑数据库配置信息

  4. 在 PHPStudy 中进入 设置 -> 配置文件 下选择你搭建 DVWA 时的PHP版本并点击，然后搜索 allow_url_include 并将其的值设置为 On 并重启 Apache/Nginx

  5. 访问 http://你建站时填写的域名或IP/setup.php ，除了 reCaptcha 之外的选项不见红即可，若有红色高亮提示则说明没配置完全，需要重复第四步（哪个红了搜哪个），配置完成后点击 Create/Reset Databse

  6. 没出意外的话，你已经安装成功了，默认帐号密码是 admin:password
